# 🔍 Python Port Scanner

## 📌 Description
This is a simple yet effective Python-based port scanner designed to identify open ports on a target system.  
It is built for learning purposes and demonstrates basic networking and socket programming concepts used in cybersecurity.

---

## 🚀 Features
- Scans ports from 1 to 1024
- Detects open ports
- Fast and lightweight
- Beginner-friendly code
- Works with both IP addresses and domain names

---

## 🛠️ Technologies Used
- Python
- Socket Programming
